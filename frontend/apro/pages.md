# 페이지 컴포넌트

> 위치: `src/pages/`

모든 페이지는 React Router의 `lazy` API로 동적 임포트되어 코드 분할이 적용됩니다. 페이지는 비즈니스 로직과 데이터 페칭을 담당하며, 실제 UI는 `components/` 하위 컴포넌트에 위임합니다.

## 인성검사 페이지 (`pages/personality/`)

세 가지 진단 유형(인성검사, 다면진단, 평판진단)은 동일한 CRUD 패턴을 따릅니다.

### 검사 작업

| 파일 | 설명 |
|------|------|
| `index.jsx` | 검사 작업 목록. 탭 필터, 검색, 페이지네이션 |
| `create.jsx` | 검사 작업 등록 폼. 검사지 선택, 기간 설정 |
| `details.jsx` | 검사 작업 상세. 탭 구조로 정보/응시자/결과 표시 |

### 검사지

| 파일 | 설명 |
|------|------|
| `sheet.jsx` | 검사지 목록 테이블 |
| `sheet-create.jsx` | 검사지 등록. 코드/문항 선택 및 구성 |
| `sheet_detail.jsx` | 검사지 상세. 구성 문항 확인/수정 |

### 코드

| 파일 | 설명 |
|------|------|
| `code.jsx` | 코드 목록 테이블 |
| `code_create.jsx` | 코드 등록 폼 |
| `code_detail.jsx` | 코드 상세/수정 |

### 문항

| 파일 | 설명 |
|------|------|
| `question.jsx` | 문항 목록 테이블 |
| `question_create.jsx` | 문항 등록 폼 |
| `question_detail.jsx` | 문항 상세/수정 |

### 데이터

| 파일 | 설명 |
|------|------|
| `data.jsx` | 데이터 내보내기/분석 |

## 다면진단 페이지 (`pages/multifaceted/`)

인성검사와 동일한 구조이며, 데이터 관리 페이지가 없습니다. 다면진단 특성상 **평가자(rater)** 관련 기능이 추가됩니다.

| 파일 | 설명 |
|------|------|
| `index.jsx` | 진단 작업 목록 |
| `create.jsx` | 진단 작업 등록 |
| `details.jsx` | 진단 작업 상세 (응시자 + 평가자 관리) |
| `sheet.jsx` | 진단지 목록 |
| `sheet-create.jsx` | 진단지 등록 |
| `sheet_detail.jsx` | 진단지 상세 |
| `code.jsx` | 코드 목록 |
| `code_create.jsx` | 코드 등록 |
| `code_detail.jsx` | 코드 상세 |
| `question.jsx` | 문항 목록 |
| `question_create.jsx` | 문항 등록 |
| `question_detail.jsx` | 문항 상세 |

## 평판진단 페이지 (`pages/reputation/`)

다면진단과 유사하며, **데이터 관리** 페이지가 추가로 있습니다.

| 파일 | 설명 |
|------|------|
| `index.jsx` | 진단 작업 목록 |
| `create.jsx` | 진단 작업 등록 |
| `details.jsx` | 진단 작업 상세 (응시자 + 평가자 관리) |
| `sheet.jsx` ~ `question_detail.jsx` | (다면진단과 동일 구조) |
| `data.jsx` | 데이터 관리 |

## 설정 페이지 (`pages/dashboard/settings/`)

`SettingsLayout`으로 감싸져 좌측 탭 네비게이션이 제공됩니다.

| 파일 | 설명 |
|------|------|
| `account.jsx` | 관리자 계정 정보 수정. 비밀번호 변경 |
| `company.jsx` | 업체 목록 테이블 (company_id 1, 2만 접근 가능) |
| `company-create.jsx` | 업체 등록 폼 |
| `company-detail.jsx` | 업체 상세. 관리자 추가/수정 모달 포함 |
| `notice.jsx` | 공지사항 목록 (company_id 1, 2만 접근 가능) |
| `notice-form.jsx` | 공지사항 작성/수정. 리치 텍스트 에디터(TipTap) 사용 |

## 공통 페이지 패턴

### 목록 페이지 패턴

```
+--------------------------------------------------+
| 브레드크럼                                         |
+--------------------------------------------------+
| [탭 필터]  [검색 입력]  [필터 버튼]  [등록 버튼]     |
+--------------------------------------------------+
| 데이터 테이블                                      |
|   체크박스 | 컬럼1 | 컬럼2 | ... | 액션            |
|   ...                                             |
+--------------------------------------------------+
| 페이지네이션                                       |
+--------------------------------------------------+
```

### 상세 페이지 패턴

```
+--------------------------------------------------+
| 브레드크럼                                         |
+--------------------------------------------------+
| [탭1] [탭2] [탭3]                                 |
+--------------------------------------------------+
| 탭별 컨텐츠 영역                                   |
|   - 기본 정보                                      |
|   - 응시자 관리 (등록/삭제/엑셀 업로드)              |
|   - 결과 확인 (다운로드)                            |
+--------------------------------------------------+
```

### 등록/수정 폼 패턴

```
+--------------------------------------------------+
| 브레드크럼                                         |
+--------------------------------------------------+
| Card                                              |
|   폼 필드들 (React Hook Form + Zod)               |
|   ...                                             |
|   [취소] [저장]                                    |
+--------------------------------------------------+
```
